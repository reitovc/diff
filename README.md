# テキスト差分比較ツール (diff)
高精度なテキスト比較ツール「difff（デュフフ）」にインスパイアされた、完全オフライン対応の単一HTMLファイルツールです。
PDFやWordからのコピペによる不自然な改行や文字のブレを吸収し、純粋な文章の差分を見やすく表示します。

https://reitovc.github.io/diff/diff_v3.html

## 特徴
* **インストール不要・完全オフライン** — 外部ライブラリやCDNを一切読み込まず、HTMLファイルをブラウザで開くだけで動作します。機密情報の比較にも安全です。
* **選べる2つの表示モード** — 1行ずつ厳密に見比べる「左右分割表示（行同期）」と、編集履歴派が見慣れている1つの文章として直感的に読む「インライン統合表示」をワンクリックで切り替え可能です。
* **スマート段落結合** — PDFコピペ時に発生する「文の途中での不自然な改行」を自動で修復し、段落として繋げてから比較します。
* **ノイズの徹底排除** — NFKC正規化により、全角/半角の違いや濁点の分離（Mac特有の現象など）を吸収。さらに、スペースやタブだけの追加・削除はハイライトせず、真の変更点のみを際立たせます。
* **ダークモード＆印刷対応** — 目に優しいダークテーマ表示、およびUIを隠して結果のみを出力する印刷（PDF保存）モードを搭載。

## 使い方
1. `diff_v3.html` をダウンロード
2. ブラウザで開く（オフライン環境でも動作します）
3. 左右のテキストエリアに変更前（Left）と変更後（Right）のテキストを貼り付ける
4. 「▶ 比較する」ボタンをクリック（または `Ctrl` + `Enter`）

## 動作環境
* モダンブラウザ（Chrome / Edge / Safari / Firefox）
* 外部通信を一切行わないため、ローカルのセキュアな環境でも動作します。

## 技術スタック
* 単一HTMLファイル（HTML / CSS / Vanilla JavaScript）
* 依存ライブラリなし
* Myers Diff Algorithm（配列・トークンベースで独自実装したO(ND)アルゴリズム）

## クレジット・謝辞
本ツールは、 @meso_cacase によってPerlで開発された高精度なテキスト比較ツール「[difff《ﾃﾞｭﾌﾌ》](https://difff.jp/)」に深くインスパイアされて作成しました。

改行を `<$>` トークンに置換して1次元配列として扱うアイデアや、装飾タグが改行を跨ぐ際のHTML構造の修復ロジックなどを活用させていただいております。素晴らしいオリジナルツールの公開に、心より感謝申し上げます。
